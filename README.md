# Hindi + Telugu Voice Bot 🎤

This project is a simple AI Voice Bot built using Python and Flask that can understand
mixed Hindi and Telugu speech, convert it into text, generate a reply, and speak the response.

This project was developed as part of a Voice Bot Assignment.

## ✨ Features

- Speech to Text (User voice → Text)
- Text to Speech (Bot reply → Voice)
- Hindi + Telugu mixed language support
- Simple and clean web UI
- Conversation display on screen
- Demo video included
- Hosted using ngrok

---

## 🛠️ Technologies Used

- Python
- Flask
- SpeechRecognition
- PyAudio
- pyttsx3
- HTML, CSS, JavaScript
- ngrok

---

## ▶️ How to Run the Project

### 1. Install required libraries

pip install SpeechRecognition pyttsx3 flask pyaudio

### 2. Run the Flask app

python app.py

### 3. Start ngrok

ngrok http 5000

### 4. Open the ngrok link in your browser.
